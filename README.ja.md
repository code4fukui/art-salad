# アート・サラダ

アートを使ったジグソーパズルをVR空間で遊ぶことができる無料のWebサイトです。

## 機能
- VR空間でジグソーパズルを遊ぶことができる
- 3Dスキャンされたアート作品やHistorical objectsを素材にしてパズルを作成
- 無料で利用できる

## 必要環境
Meta Questなどの仮想現実(VR)デバイスが必要です。

## 使い方
1. VRデバイスを装着する
2. ウェブブラウザを起動する 
3. https://artsalad.net にアクセスする
4. 「VRに入る」をクリックする
5. ジグソーパズルを楽しむ

## データ
パズルのモデルの大部分は[Sketchfab](https://sketchfab.com/)から提供されており、Creative Commonsライセンスで公開されています。
