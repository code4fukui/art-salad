# アート・サラダ

アートを使ったジグソーパズルをVR空間で楽しめる無料のWebサイトです。

## 機能
- VR空間でジグソーパズルを遊ぶことができる
- 3Dスキャンされたアート作品やHistorical objectsを素材にパズルを作成
- 無料で利用できる

## 必要環境
MetaのQuestなどのVRデバイスが必要です。

## 使い方
1. VRデバイスを装着する
2. ウェブブラウザを起動する
3. https://artsalad.net にアクセスする
4. 「VRに入る」をクリックする
5. ジグソーパズルを楽しむ

## データ
パズルのほとんどのモデルは[Sketchfab](https://sketchfab.com/)から提供されており、Creative Commonsライセンスで公開されています。
