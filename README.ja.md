# アート・サラダ

アートを使ったジグソーパズルをVR上で楽しめる無料のWebサイトです。

## デモ

VRデバイス（Meta Questなど）を使えば、ウェブブラウザで直接アクセスできます。
[https://artsalad.net](https://artsalad.net)

## 機能

- 3D スキャンされた芸術作品をジグソーパズルとして遊べる
- VR空間で没入体験ができる
- 無料で利用できる

## 必要環境

- VRデバイス（Meta Questなど）
- ウェブブラウザ

## 使い方

1. VRデバイスを装着する
2. ウェブブラウザを起動する
3. [https://artsalad.net](https://artsalad.net) にアクセスする
4. 「VRに入る」ボタンをクリックする
5. ジグソーパズルを楽しむ
